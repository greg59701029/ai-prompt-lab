# v0.4.0

AI Prompt Lab v0.4.0 turns the project from a static prompt builder into a more maintainable browser tool with reusable presets, live deployment, stronger tests, and clearer accessibility signals.

## Highlights

- Live GitHub Pages demo
- JSON preset import and export
- Shared prompt core module
- Unit tests for prompt generation, scoring, stats, and preset parsing
- Playwright E2E tests for template selection and preset workflows
- README screenshot and status badges
- Accessibility improvements for status updates, generated prompt output, and quality meter

## Validation

- `python tests/smoke_test.py`
- `npm run test:unit`
- `npm run test:e2e`
- GitHub Pages deployment
