# v0.5.0

AI Prompt Lab v0.5.0 closes the first small maintenance milestone after the static release. It focuses on real browser workflows: saved prompt setups, a learning template, prompt diffing, and stronger accessibility-oriented coverage.

## Highlights

- Browser-local named preset save, load, and delete actions
- Learning prompt template for study plans and interview preparation
- Baseline prompt diff view for changed prompt lines
- Visible checklist state labels for accessibility
- Keyboard workflow coverage for copy, template selection, import, and export
- Playwright coverage for named preset management

## Validation

- `python tests/smoke_test.py`
- `git diff --check`
- GitHub Actions CI
- GitHub Pages deployment
