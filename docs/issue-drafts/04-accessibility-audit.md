# Improve accessibility coverage

## Acceptance Criteria

- Add keyboard-only workflow coverage for template, copy, import, and export actions
- Confirm status updates are announced through `aria-live`
- Confirm color is not the only signal for quality checklist status

- Document accessibility expectations in maintainer notes
